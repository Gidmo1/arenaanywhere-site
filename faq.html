FAQ

Questions people actually ask.

If it’s not here, it’s probably a skill issue. Just kidding — we’ll add it.

Do I need to install Tailscale?

No. ArenaAnywhere handles the connection setup.

Is it free?

Yes. Core matchmaking + connect is free.

Which games are supported?

Any ad-hoc game, optimized for PES/eFootball.

What if I can’t connect?

Use the built-in connection test + support tips inside the app.

Will this reduce lag?

It won’t magically delete your network issues — but it helps you connect reliably. Lag depends on distance, network quality, and device performance.

Is it safe?

The goal is simple: connect you to your opponent for the match. No sketchy “install 5 apps” nonsense.

What Android versions are supported?

We’re targeting modern Android versions first. Early access users will get exact requirements as builds drop.

Do both players need ArenaAnywhere?

Yes — for matchmaking and auto-connect to work cleanly, both players should use it.

Still stuck?

No shame. Some networks are just… evil.
